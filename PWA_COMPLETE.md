## 📱 How Users Install

### iPhone/iPad:
1. Open Safari → Go to your deployed URL
2. Tap Share button → "Add to Home Screen"
3. Done! App appears on home screen

### Android:
1. Open Chrome → Go to your deployed URL
2. Tap "Install" banner or Menu → "Install app"
3. Done! App appears on home screen

### Desktop:
1. Open in Chrome/Edge → Click install icon in address bar
2. Or Safari (Mac) → Share → Add to Dock
3. App opens in its own window!

Want to change the app icon or colors?

1. **Icons**: Edit `generate-icons.js` and run `node generate-icons.js`
2. **Colors**: Edit `public/manifest.json` and `index.html` theme-color
3. **Name**: Edit `public/manifest.json`

See [PWA_GUIDE.md](./PWA_GUIDE.md) for details.
